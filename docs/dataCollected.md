# Data collected for accounts 
- User ID (id): a random uniqute identifier used to identify user.
- Username (IGN): pseudonyms or nicknames used for login and to show ownership of contributions.
- hashPassword: gibberish text derived from the original password.
- ToSAcceptedAt: time when the Terms of Service was accepted.
- consentVersion: version of the document


# Data collected for content - Public Contribution
- AuthorId: links to a chapter/branch to the account that created it.


## Data that are NOT needed to be collected
- Any private data that are connected to the author are NOT collected, for example email, phone number, etc.
